1.1.1. Скачивание и установка Python

Python — это интерпретируемый язык программирования высокого уровня. Для начала работы необходимо установить интерпретатор Python с официального сайта python.org. Рекомендуется использовать версию 3.10 или выше.

Установка на Windows:

  • Скачайте установщик с python.org (выберите версию для Windows)
  • Запустите установщик
  • ВАЖНО: отметьте галочку "Add Python to PATH"
  • Нажмите "Install Now"

Установка на macOS:

  • Через Homebrew: brew install python
  • Или скачайте установщик с python.org

Установка на Linux:

  • Ubuntu/Debian: sudo apt update && sudo apt install python3
  • Fedora: sudo dnf install python3
💡 ПРОВЕРКА УСТАНОВКИ: Откройте терминал (командную строку) и выполните команду python --version или python3 --version. Должна отобразиться версия Python.

Проект 1: To-Do менеджер

Консольное приложение для управления списком задач. Позволяет добавлять, отмечать выполненные, удалять и просматривать задачи. Данные сохраняются в JSON файл.

📁 Структура проекта

todo_manager/
├── main.py          # основной файл
├── tasks.json       # файл с данными (создаётся автоматически)
└── README.md        # описание проекта

📄 Код программы (основная часть)

import json
import os
from datetime import datetime

class TodoManager:
    """Класс для управления списком задач"""
    
    def __init__(self, filename="tasks.json"):
        self.filename = filename
        self.tasks = self.load_tasks()
    
    def load_tasks(self):
        """Загружает задачи из файла"""
        if os.path.exists(self.filename):
            with open(self.filename, 'r', encoding='utf-8') as f:
                return json.load(f)
        return []
    
    def save_tasks(self):
        """Сохраняет задачи в файл"""
        with open(self.filename, 'w', encoding='utf-8') as f:
            json.dump(self.tasks, f, ensure_ascii=False, indent=2)
    
    def add_task(self, title):
        """Добавляет новую задачу"""
        task = {
            "id": len(self.tasks) + 1,
            "title": title,
            "completed": False,
            "created_at": datetime.now().strftime("%d.%m.%Y %H:%M")
        }
        self.tasks.append(task)
        self.save_tasks()
        print(f"✓ Задача '{title}' добавлена")
    
    def show_tasks(self):
        """Показывает список задач"""
        if not self.tasks:
            print("Список задач пуст")
            return
        
        print("\n" + "="*50)
        print("СПИСОК ЗАДАЧ:")
        print("="*50)
        for task in self.tasks:
            status = "✓" if task["completed"] else "○"
            print(f"{status} [{task['id']}] {task['title']}")
        print("="*50)
    
    def complete_task(self, task_id):
        """Отмечает задачу как выполненную"""
        for task in self.tasks:
            if task["id"] == task_id:
                task["completed"] = True
                self.save_tasks()
                print(f"✓ Задача '{task['title']}' выполнена")
                return
        print(f"Задача с ID {task_id} не найдена")

Задание №16: Рекурсивные алгоритмы

📌 ТИПИЧНАЯ ФОРМУЛИРОВКА

Алгоритм вычисления функции F(n) задан рекуррентными соотношениями. Найдите значение F(5) или определите количество вызовов функции.

Теоретическая основа

Рекурсия — это способ определения функции через саму себя. Для решения задач №16 необходимо:

  • Определить базовые случаи (условия остановки рекурсии)
  • Построить дерево рекурсивных вызовов
  • Вычислить значение последовательно

Пример решения

Условие: Алгоритм вычисления функции F(n) задан следующими соотношениями:

  • F(n) = n при n ≤ 2
  • F(n) = F(n-1) + 2*F(n-2) при n > 2

Чему равно значение F(6)?

from functools import lru_cache

@lru_cache(maxsize=None)
def F(n):
    if n <= 2:
        return n
    return F(n-1) + 2 * F(n-2)

print(F(6))  # 32

Пошаговый расчёт

  • F(1) = 1
  • F(2) = 2
  • F(3) = F(2) + 2×F(1) = 2 + 2×1 = 4
  • F(4) = F(3) + 2×F(2) = 4 + 2×2 = 8
  • F(5) = F(4) + 2×F(3) = 8 + 2×4 = 16
  • F(6) = F(5) + 2×F(4) = 16 + 2×8 = 32
💡 СОВЕТ: Используйте декоратор @lru_cache для ускорения рекурсивных вычислений. Он запоминает результаты вызовов и не вычисляет их повторно.
📌 ЗАДАЧА ДЛЯ САМОСТОЯТЕЛЬНОГО РЕШЕНИЯ:
F(n) = n при n ≤ 3
F(n) = F(n-1) + 2×F(n-2) - F(n-3) при n > 3
